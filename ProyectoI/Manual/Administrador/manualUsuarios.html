Pantalla principal:

Al iniciar la aplicación, se muestra la pantalla principal de la gestión de usuarios. Esta pantalla permite agregar, modificar y eliminar usuarios, así como consultar la lista de usuarios registrados en el sistema.

Funcionalidades:

1. Agregar un usuario

Para agregar un usuario, siga estos pasos:

  1. Inicie la aplicación.
  2. Ingrese los datos en cada campo de texto (usuario, contraseña) proceda a seleccionar el tipo de usuario (Administrador, Colaborador) y el estado del usuario.
  3. Haga clic en el botón "Agregar".

Advertencias

  • No sera posible añadir un usuario con un nombre ya existente.
  • No sera posible añadir un usuario con el nombre: admin.
  • No sera posible añadir un usuario si no se ingresa alguno de los datos (usuario, contraseña, tipo de usuario, estado del usuario).

    • 2. Consultar un usuario

    Para consultar un usuario, siga estos pasos:

    1. Inicie la aplicación.
    2. Si desea consultar la información de todos los usuarios en los datos registrados,simplemente haga clic en el botón "Consultar".
    3. Si desea consultar la información de un usuario en especifico, proceda a ingresar el nombre del usuario en el campo de texto (usuario).
    4. Haga clic en el botón "Consultar".

    Advertencias

  • Si se introduce el nombre de un usuario que no existe, no se mostrara ninguna información.

    • 3. Modificar un usuario

    Para modificar un usuario, siga estos pasos:

    1. Inicie la aplicación.
    2. Ingrese los datos que desea modificar en cada campo de texto (usuario, contraseña).
    3. Si desea modificar el tipo de usuario (Administrador, Colaborador) o el estado del usuario (Activo, Inactivo) seleccionelo.
    4. Haga clic en el botón "Modificar".

    Advertencias

  • Si se introduce el nombre de un usuario que no existe, no se modificara ningun usuario
  • Si no se introduce o selecciona un dato, se procedera a preguntar si desea modificar ese dato, si no lo desea se la información.

    • 4. Eliminar un usuario

    Para eliminar un usuario, siga estos pasos:

    1. Inicie la aplicación.
    2. Si desea eliminar un usuario, introduzca el nombre del usuario a eliminar en el campo de texto (usuario)
    3. Haga clic en el botón "Eliminar".

    Advertencias

  • No es posible eliminar un usuario que no existe
