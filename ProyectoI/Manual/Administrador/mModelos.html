Pantalla principal:

Al iniciar la aplicación, se muestra la pantalla principal de la gestión de modelos. Esta pantalla permite agregar, modificar y eliminar modelos de aviones, así como consultar la lista de modelos registrados en el sistema.

Funcionalidades:

1. Agregar un modelo de avión

Para agregar un modelo de avión, siga estos pasos:

  1. Inicie la aplicación.
  2. Ingrese los datos en cada campo de texto (modelo, asientos ejecutivos, asientos turistas, asientos económicos) y seleccione la marca del avión.
  3. Haga clic en el botón "Agregar".

Advertencias

  • No será posible agregar un modelo de avión con un nombre ya existente.
  • No será posible agregar un modelo de avión si no se ingresan todos los datos (modelo, asientos ejecutivos, asientos turistas, asientos económicos y marca del avión).

    • 2. Consultar un modelo de avión

    Para consultar un modelo de avión, siga estos pasos:

    1. Inicie la aplicación.
    2. Si desea consultar la información de todos los modelos de aviones registrados, simplemente haga clic en el botón "Consultar".
    3. Si desea consultar la información de un modelo de avión específico, ingrese el nombre del modelo en el campo de texto (modelo).
    4. Haga clic en el botón "Consultar".

    Advertencias

  • Si se introduce el nombre de un modelo de avión que no existe, no se mostrará ninguna información.

    • 3. Modificar un modelo de avión

    Para modificar un modelo de avión, siga estos pasos:

    1. Inicie la aplicación.
    2. Realice una consulta del modelo de avión que desea modificar, siguiendo los pasos mencionados anteriormente.
    3. Ingrese los que desea modificar en cada campo de texto (modelo, asientos ejecutivos, asientos turistas, asientos económicos) y la marca del avión.
    4. Haga clic en el botón "Modificar".

    Advertencias:

  • No será posible modificar el nombre del modelo de avión.
  • Si no se introduce o selecciona un dato, se procedera a preguntar si desea modificar ese dato, si no lo desea se la información.

    • 4. Eliminar un modelo de avión

    Para eliminar un modelo de avión, siga estos pasos:

    1. Inicie la aplicación.
    2. Realice una consulta del modelo de avión que desea eliminar, siguiendo los pasos mencionados anteriormente.
    3. Una vez que se muestre la información del modelo de avión, haga clic en el botón "Eliminar".
    4. Se mostrará una confirmación de eliminación, donde deberá confirmar su acción.
    5. Una vez confirmado, el modelo de avión será eliminado del sistema.

    Advertencia:

  • No pueden eliminarse modelos que hayan sido asociados a algún avión.
  • Una vez eliminado, no se podrá recuperar la información del modelo de avión.
